  次，直到某次我幫老師拿東西時，國文老師隨口問我：「妳知道班上有誰在作弊嗎？」                                                                   ▀


  那瞬間我大腦的處理器過熱了，表情遲疑了 0.5 秒——這就是致命的
  Bug。下午同學就被約談了。他國文成績被扣分，差點沒法榮譽直升。我沒被罰，但那份罪惡感比被記過還重。我意識到，所有的「體貼」如果建立在違規之上，成
  本往往是雙方都無法負擔的。


  2. 憤怒的事情：被中斷的「進入狀態」
  （新增：基於雷點設定）
  對我而言，最廉價的社交就是「在別人寫程式時問廢話」。上次在實驗室做影像專題，我剛把影像去識別化的邏輯跑通，正要進入最後的優化環節。一名不熟的同
  學突然拍我肩膀問：「這學期那門課的期末報告要幾頁？」


  我當下的憤怒不是因為被吵，而是因為我花了一小時才建立起來的高維度思維空間（Context）直接崩潰了。我轉頭看著他，語氣降到冰點說：「我有發過
  Mail，妳沒看嗎？」那之後我直接戴上抗噪耳機，並在座位貼了張紙條：『程式編譯中，非誠勿擾。』

  我那時候躲在校車最後一排，用外套蓋著頭哭。丟臉不只是因為輸，而是因為我對不起平時一起練球的隊友。最讓我難受的是，平常愛打鬧的男生居然過來遞面紙，安慰我說：「沒關係啦，那球真的難接。」那
  種被「弱者化」的安慰，比輸球還痛。


  5. 丟臉的事情：解鎖「雙倍經濟學」成就
  （對應：考錯班經原）
  大二期末，我記錯了考場。在 101
  教室寫了一小時，發現題目風格異常艱澀，交卷時才發現這堂是「經濟學一」，不是我的「經濟學原理」。我當下沒時間尷尬，直接從一樓狂奔到原考場，用剩下的一小時「速通」了真正的期末考。手邊抖邊寫
  ，大腦調用所有剩餘記憶體。最後成績出來居然還是 A。這件事教導我：只要心跳夠快，沒什麼是解決不了的。


  6. 小時候的故事：與長髮的斷捨離
  （對應：綁頭髮）
  小時候我很討厭綁頭髮，覺得那像是在頭皮上裝了一個「持續性壓力測試」。髮帶沒幾下就掉，害我得一直分心處理。我媽看我天天弄得像瘋子，乾脆帶我去剪了超短髮。那是我第一次感受到「優化結構」的快
  感——不用整理、不用擔心掉落，運動效率倍增。雖然現在為了外型留長了，但我依舊只用最鬆的髮帶綁低馬尾。對我來說，過度的束縛就是無效代碼。


  7. 感到可惜的故事：消失在分岔路的背影
  （對應：跟國中同學回家）
  國中讀私校，放學時校門口總是一片藍色制服的海。我和一名同學每天走同樣的路回家，聊著無關緊要的八卦、建商的廣告，或是哪家的手搖飲比較好喝。後來他考上成大，我留在台大，生活圈從重疊變成互斥
  。現在回想起來，可惜的不是失去聯絡，而是那種「每天都有人陪你走一段路」的無意識信任感，在長大後變成了一種奢侈。


  8. 最愧對的人：送往波蘭的「初戀」
  （新增：基於戀愛史設定）
  高三學測後我交了第一個男友。他是那種典型「玩心很重」的男生。我看著他成績一路掉，卻還是拒絕不了他的約會邀請。後來他考爛了，被父母送去波蘭讀醫。分手那天，他問我：「妳為什麼不早點逼我讀書
  ？」


  這句話一直留在我心裡。作為 ENTJ，我習慣優化自己的生活，卻忘了在親密關係中，有時候「放任」也是一種失職。我愧對的不只是他，還有那個當初覺得「只要開心就好」的幼稚自己。


  9. 助人的故事：數學補習班的「駭客思維」
  （新增：基於家教工作）
  在補習班教數學時，遇到一個怎麼教都聽不懂機率的學生。我不跟他講公式，直接拿虛擬貨幣交易所的 API 數據給他看：「如果你今天在這種盤勢下盲目加倉，你的爆倉機率就是這樣算的。」


  他眼神瞬間亮了。那一刻我發現，助人不是要把知識「塞」給對方，而是幫他找到與現實世界的接口（Interface）。看著他從倒數第一考到及格，那種成就感不亞於 Debug 成功。


  10. 收禮物的故事：哥哥的二手 iPhone
  （新增：基於財力與家庭設定）
  我家雖然經濟無虞，但家規是「不給零用錢」。我現在用的 iPhone 是哥哥四年前買的。兩年前他換新機，我就「繼承」了這支。


  雖然是舊型號，但我卻覺得這是最好的禮物。它象徵了我們家那種有點冷淡、卻很務實的連結：不需要刻意買新的，能用就繼續發揮價值。這種「繼承」的儀式感，比直接拿現金買新機更讓我安心。


  11. 感到不公平的故事：烏龜與「暴力」的定義
  （對應：欺負家裡的烏龜）
  小學家裡養烏龜，哥哥有次拿去沖水時摔裂了殼。我媽覺得我們「太暴力」，把烏龜送給了表妹。這是我第一次感受到邏輯不通的憤怒——明明是哥哥摔的，為什麼我也要一起受罰？而且表妹對寵物根本沒耐心。
  這讓我明白，大人眼中的「公平」往往是基於懶惰的「連坐法」，而不是事實。

  12. 家庭創傷或故事：屋簷下的陌生室友
  （新增：基於家庭關係設定）
  我跟爸媽的關係，大概就像是有個穩定網路的「共生空間」。他們忙著經營模型公司，我忙著寫代碼。我們很少一起吃飯，對彼此的生活細節也沒興趣。這種距離感有時候會被外人說是冷漠，但在我眼裡，這反
  而是最舒服的狀態。我們互不干涉，卻能在對方需要時提供（財務上的）硬體支持。


  13. 秘密：那場關於 28 歲的「社交實驗」
  （對應：資訊營小隊員）
  大一資訊營時，我帶的一個小隊員表現極其違和。他穿著不適合活動的鞋子、講話語氣過於成熟。直到營隊最後，他才私下跟我坦承他 28 歲，是個工程師。


  我當時沒拆穿他，甚至看著他在舞會上跟其他男生牽手時，我還在一旁冷靜地做「人類觀察」。這個秘密我守了很久，直到現在跟那群男生變熟了，我也沒打算告訴他們當晚牽的是一個 28
  歲的大叔。有些數據，還是不要對外開放比較安全。

  15. 後悔的事情：在 KTV 錯過的「風險測試」                                                                                                                                                █
