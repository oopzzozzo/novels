# 考錯班經原
發文來紀念人生發生過最蠢的事
今天早上 10:20 考經原在101
我記錯成201，準時進去考試
考卷發下來時發現風格跟第一次考試差很多
覺得很怪但還是把它寫完 11:20 提早交卷
考卷上寫說第三次期末考
我印象中我只考過一次期中
於是交卷時我問了監考的助教
助教說他有寄信跟大家說
但我說我沒收到
我把我收到經原助教寄的信給監考助教看
他看完跟我說：我們這堂課是「經濟學一」不是經原
這時我才發現我愚蠢的考錯考卷了
只好狂衝到原班考試
在剩下一個小時內邊抖邊寫考卷 QAQ
慶幸我最後有寫完
解鎖新成就：兩個小時考兩份經濟學期末考
# 資訊營小隊員
我大一暑假擔任資訊營隊輔，有一位小隊員很難搞。穿不好走的鞋子，導致腳受傷。晚上說弟弟失戀要去外面陪講整晚的電話。結果最後營隊結束前，他才坦誠自己「已 28 歲，現職軟體工程師。十年前參加過資訊營覺得感動，於是想來重溫。」我對舞會和他牽手的男生們感到非常抱歉。
# 體育加簽
我小學是桌球校隊，但是現在沒在打。體育課都挑不是早八的，像是手球和橄欖球。大二開學那天中午，系桌同學拉我去加簽桌球初級。「聽說桌初的加簽規則是所有同學排三排報數。同數的同學猜拳，贏者可以拿到授權碼。」「我們一起去加簽好不好？我出剪刀你出布。我請你吃小福冰淇淋。」我們去體育館的路上討論了我們的配拳演算法、為了避免對齊失敗大家要排在最前面等等策略。
結果體育館出現大量加簽人潮，助教先是叫大家排五排，又改口要排七排。一片混亂後，大家總算排好坐下。老師：「人有點多。排頭站起來，你們圍一圈猜拳，贏的那整排發授權碼。」我站著好緊張，看著另外六人中的三個自己人，心裡默念配拳順序。總算猜到第四把時，其他人剛好都輸給我們，我才鬆了口氣。
輸完回系館的路上，餘悸猶存。系桌同學真是計畫通。
# 幫同學改考卷
國八中時早上都有週考，坐我右邊的同學注釋總是背不起來，我們說好他先用鉛筆作答，交換改的時我都打勾給分，他回去再抄答案。
這樣過完第一次段考。有天數學老師叫我到辦公室幫忙拿東西，卻遇到國文老師問我知不知道有人考試作弊。我嘴上說不知道，表情卻心虛了一下被發現。下午同學就被約談了，他整學期國文成績被扣五分，差點不能榮譽直升。
# 教會
小三時的聖誕節，同學邀請我去看教會的聖誕表演，結果前面一直坐著講古講好久，還要一起念經文不能聊天。我去過一次以後從此就不去了。現在在學校遇到傳教的人，都覺得他們真有耐心。
# 拒毒
高二一個週六下午放學後，我和班上幾個比較自由的同學去 KTV。田僑仔同學在水煙裡面加料。大家都很好奇，我也想試試。但轉念一想，萬一我真的成癮，仰賴這位同學供貨好像不太可靠，就嫌臭拒絕了。不過後來也沒聽說常用的同學出事。
# 綁頭髮
我很討厭綁髮，簡直是頭皮戴緊箍。因為小時候髮帶沒兩下就掉了，所以小學時被媽媽帶去剪短髮。現在長髮有點波浪，每當它亂到必須綁起來時，我就會想趕緊去離子燙。
# 桌球隊
小六時的全國賽團體組，我打混雙打點，因為一直發球失誤，害整隊第一場就被淘汰。在車上一個人縮著哭，回學校被隊友發現安慰我。居然被當時心智年齡超低的隊友安慰，現在想起來真是丟臉。
# 上學遲到穿拖鞋
我高中家住學校附近，所以都走路上學。俗話說住得越近的人越常遲到，就是在說我。但是我會注意不要為難教官。早上七點半算遲到，我如果七點半起床的話，七點四十五去學校遇到教官會很為難。所以我早上起來會視情況做個早餐慢慢吃完，八點後再穿拖鞋和便服，跟警衛打招呼後進校門。這也是為了校譽著想，沒有人想看到學校有遲到的學生。
# 欺負家裡的烏龜結果被送走
小學時哥哥同學送了一隻烏龜，我們都給牠吃高麗菜，把他放到床底下檢查床底下有沒有鬼。有一次哥哥拿牠去沖水的時候，把牠的殼摔了一小小角。媽媽覺得我們太爆力，就把牠送給表妹了。明明表妹就比哥哥還要爆力。
# 跟國中同學回家
因為國中讀私校，大部份的同學都坐校車，很少走路上學。我和一位同學幾乎每天都一起回家。我們一起看了各種牽手去搭公車的學長姐、在教官引導下過馬路的小學生、建商的預售屋廣告…但是他高中讀乙班，大學讀成大，就沒有天天見面了。
